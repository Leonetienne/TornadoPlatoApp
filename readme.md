# TornadoPlato

This is still under heavy development. It's just public for the fun in it.

## How2cook
1. Haf cmake installed
2. `cd _Testapp`
3. `cmake -B build`
4. `cd build`
5. `make`
6. `./_Testapp.out`

## How2cook SDL2 application
1. Haf cmake installed
2. Haf SDL2 installed
3. `cd _TestappSDL2`
4. `cmake -B build`
5. `cd build`
6. `make`
7. `./_TestappSDL2.out`
