OPTIMIZATIONS:

	GAME ENGINE:
	Support culling distances for light sources and mesh renderers

	RENDERING ENGINE:
	Cull lights if distance to camera is > farclip + intensity*13


FEATURES:
	SIMPLE LIGHT SOURCES:
		Attribute: Box boundingBox
			If boundingBox is set, this light source should only cast light within the box.
			User story: We want to use a shadowless light to light a house from the inside.
				    We want to avoid the light casting brightness onto objects outside the house.
			            The house would be the bounding box.
